# react-native-b-crypt-bridge
React Native package to use b-crypt functions
## Installation

```sh
npm install react-native-b-crypt-bridge
```

## Usage

```js
import { multiply } from "react-native-b-crypt-bridge";

// ...

const result = await multiply(3, 7);
```
